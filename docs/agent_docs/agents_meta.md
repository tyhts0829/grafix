# AGENTS のメタ方針

## 原則

- WHY/WHAT/HOW を短く置く（重要情報ほど冒頭）。
- 指示は最小限にし、正否は `pytest` / `ruff` / `mypy` で判定する。
- タスク固有の詳細は `docs/agent_docs/` に分離し、ルート `AGENTS.md` にはポインタだけ置く。
- 迷ったら明確さ・シンプルさを優先し、過度に防御的にしない。
- 「報告/提案」系の依頼では、聞かれた範囲だけ答え、先回りで変更しない。
