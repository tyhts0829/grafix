# どこで: `README.md` 冒頭の GIF 表示
# 何を: README 先頭の `readme.gif` を「横幅いっぱい（カラム幅いっぱい）」に表示されるようにする
# なぜ: 作品デモの視認性を上げたい（サムネイル的に小さく見えてしまうのを避ける）

作成日: 2026-01-19

## ゴール

- GitHub 上の `README.md` で、冒頭 GIF がカラム幅いっぱいに表示される。

## 非ゴール（今回やらない）

- GIF 自体の作り直し（解像度/内容の変更）。
- README 全体のレイアウト調整（見出し・余白など）。

## 方針

- Markdown の `![](...)` を HTML の `<img>` に置き換える。
- `width` を大きめの px 値に指定し、GitHub 側の `max-width: 100%` と組み合わせて「常にカラム幅いっぱい」になるようにする。
